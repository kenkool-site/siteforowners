# Demo Marketing Assets

Place final `/demo` showcase media in this folder.

Recommended first assets:

- `demo-reel.mp4` - main hero reel, 20-40 seconds, muted loop.
- `demo-reel-poster.jpg` - still frame fallback for the hero reel.

Optional portfolio card assets:

- `nails-preview.mp4`
- `locs-preview.mp4`
- `lashes-preview.mp4`
- `barber-preview.mp4`
- `spa-preview.mp4`

Keep clips short, mobile-friendly, and free of private customer details.
